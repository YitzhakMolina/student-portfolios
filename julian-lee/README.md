# Description

I took this opportunity to learn a lot about something that I've been interested in recently. My video tries to show how it's possible for a machine to learn by itself just by analyzing data in neural networks. It goes over some terminology definitions, who first thought of the idea to create a brain inspired computer sequence, and the whole neural network chain from start to finish. If I had to summarize everything really shortly, it'd be that you feed an algorithm made up of hundreds to millions of matrices some data in the form of a matrix, and it outputs a final matrix that is supposed to be some kind of interpreted version of that input matrix you put it. Then it compares its interpretation to what it's actually supposed to be to see how close it got to getting it right. Then it uses calculus to work backwards in order to find and correct each element in every matrix that contributed to the error. Then it just repeats.

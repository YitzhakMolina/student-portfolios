**📌 About the Project**

This project looks at how Artificial Intelligence (AI) and Machine Learning are changing the way we analyze data.

To show how this works in the real world, the project compares two different fields:

Actuarial Science: How insurance companies use AI to look at financial risk (like tracking driving habits to set insurance prices).

Cybersecurity: How security teams use AI to protect computer networks (like detecting suspicious logins and stopping hackers).

The project explains that while AI is a great tool for finding patterns and making predictions quickly, we still need human experts to make ethical choices and final decisions.

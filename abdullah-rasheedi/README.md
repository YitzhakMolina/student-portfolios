
My presentation aims to give a brief overview of AI. I start off by explaining how AI is different from conventional programming, and then move on to mentioning how it works similarly to how our brain functions. I also mention some differences between our brains and an AI's neural network. Then we talk about how artificial neurons take weighted sums and add a bias, and provide an output. These neurons are layered to form a neural network, which aims to replicate how we make abstractions of information to highlight patterns in data. A network doesn't just have random weights and biases, though, because there is no logic in that, but it starts as such, so I explain the process of gradient descent. Finally, we explore transformers and attention mechanisms because a pattern recognition machine must be supplemented with technology to curate inputs to understand things such as context. The final page has citations.
